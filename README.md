## Aliases
- `zshrc` -> `nano ~/.zshrc`
- `la` -> `ls -A`
- `ll` -> `ls -l`
- `lla` -> `ls -l -A`
- `sdnfi` -> `sudo dnf install`
- `dnfi` -> `dnf install`
- `dnfs` -> `dnf search`
- `sdnfu` -> `sudo dnf upgrade`
- `dnfu` -> `dnf upgrade`
